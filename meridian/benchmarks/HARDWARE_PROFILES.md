# Hardware Profiles

Last updated: 2026-03-23

Fill this file as benchmark hardware is chosen.

## Required Profiles

### Profile A: NVIDIA desktop

Record:

- GPU
- VRAM
- CPU
- RAM
- OS
- driver version
- Vulkan version
- display resolution

### Profile B: AMD desktop

Record:

- GPU
- VRAM
- CPU
- RAM
- OS
- driver version
- Vulkan version
- display resolution

### Profile C: older or fallback desktop

Record:

- GPU
- VRAM
- CPU
- RAM
- OS
- driver version
- Vulkan version
- display resolution

Goal:

- at least one modern NVIDIA path
- at least one modern AMD path
- at least one non-ideal fallback path
