# Godot Integration

Purpose:

- importer integration
- custom `VGeoMesh` resource support
- editor tooling and debug UX
- eventual runtime scene integration

This directory should not be treated as the primary place to prove renderer feasibility.
The standalone prototype is the first runtime proving ground.

Planned contents:

- `src/`: importer, resource, and integration code
- `shaders/`: Godot-side shader assets if needed
- `demo/`: small integration test scenes
